The Interface

Viewing Database Information

RAW database information is located at: DB/database.script

To view Database information, press the Refresh button

Add Information to the Database

To add information to the database, click the add button

This will open the menu to add information to the database. Clicking an item from the "add" menu list will open a new window to input desired data.

Delete from database

To delete from the database, select the row you wish to delete and click the delete button.

This will permanetly delete the row and all of the associated information from the table.

Search and view wedding information

To search the database for a specified wedding, click the magnifying glass.

This will display a menu with various search requests.


The search menu

Search Weddings displays the search view. You can search a wedding by entering in the wedding name. Note, the wedding name does not have to be exact. The query selects the closest match to the specifed search, but the first letter must be capitalized for accurate results.


The search window

The above example displays a substring of the wedding Randy & Lahey.

The search returns the specified wedding name, a list of items used at the wedding, a count of the individual item, and the various items vendor name and price. On the left side of the window, the search will return the weddings date, price of the wedding, and if the wedding has been paid or not.

Methods

Getting the Data

The data was parsed using a java application that randomly generated the queries and data to a file.

Displaying the Data

Database schema and table data were developed inside of HSQLDB and saved as a .script file.

The GUI

The GUI was developed utilizing the Model View Controller approach. The models hold the table columns getter and setter methods. These methods were retrieved by an observable list that updates whenever the database inputs or outputs data.
